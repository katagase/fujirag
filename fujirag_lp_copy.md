### Q&A一覧

#### 導入・環境について

**Q1. 今使っているNASをそのまま使えますか？**
```
A. はい。既存のNASに接続するだけで動作します。
   ファイルの整理・移行・再分類は一切不要です。
```

**Q2. クラウドに移行する必要はありますか？**
```
A. ありません。FUJI RAGは完全オンプレミスで動作します。
   データは社外に一切出ません。
```

**Q3. 導入にはどれくらいの期間がかかりますか？**
```
A. PoCは1〜2ヶ月、本番導入まで含めると3〜6ヶ月が目安です。
   まずはPoC（小規模試験）から始めることをお勧めしています。
```

**Q4. 対応しているファイル形式は？**
```
A. PDF・Word（.docx）・Excel（.xlsx）・PowerPoint（.pptx）・
   テキストファイル・ソースコードに対応しています。
   画像PDFや手書き文書はOCR機能で対応します。
```

---

#### セキュリティ・権限について

**Q5. 社外にデータが出ることはありませんか？**
```
A. ありません。FUJI RAGは完全オンプレミス運用のため、
   すべてのデータが社内サーバーに留まります。
```

**Q6. 部門ごとに見れる情報を制限できますか？**
```
A. はい。Active Directoryと連携し、現在のファイルサーバーの
   権限設定をそのままAIに引き継ぎます。
   部門・役職・個人単位での細かなアクセス制御が可能です。
   権限のないフォルダの内容は、AIも回答に含めません。
```

**Q7. AIが権限外の情報を漏らすことはありますか？**
```
A. ありません。FUJI RAGは回答生成時にAD権限を確認し、
   アクセス権のないドキュメントは回答ソースから除外します。
```

---

#### 機能・精度について

**Q8. AIの回答は正確ですか？根拠はどこで確認できますか？**
```
A. すべての回答に出典（ファイル名・フォルダパス・ページ番号）を明示します。
   社内文書の範囲内でのみ回答するため、根拠のない回答（ハルシネーション）を
   大幅に低減しています。
```

**Q9. ファイルが更新されたら、AIの回答も変わりますか？**
```
A. はい。差分クロール機能により、ファイル更新を自動で検知し、
   AIの回答ソースを最新状態に保ちます。手動更新は不要です。
```

**Q10. 検索精度はどれくらいですか？**
```
A. 社内文書の内容・構造に最適化されたインデックスを構築しています。
   PoCにて実際の社内データで精度検証を行った後に本番導入を判断いただけます。
```
